# Problem 8 – Smart Navigation System (BFS, DFS)

Overview
Smart Navigation System is a simple path-finding AI application which finds a path between two graph nodes. In other words, a graph is a set of places linked by routes, which allows to define a start and a finish location. The algorithm tries to find a path through them. This task represents how AI can be used for solving real-life issues, such as finding a route on Google Maps or GPS navigators, moving around robots, or even computer networking.

Objective
To develop a graph with connected places;
To accept starting and ending nodes from the user input;
To conduct searches with help of certain algorithms;
To evaluate BFS and DFS algorithms for their differences in performance.

Algorithms Used
BFS (Breadth-First Search)
- Finds a route among all routes in breadth;
- Advantage: the shortest way in case of unweighted graphs;
- Disadvantage: requires higher memory consumption.

DFS (Depth-First Search)
- Works by visiting one branch completely;
- Advantage: simpler realization; can consume less memory than BFS;
- Disadvantage: does not guarantee the shortest way.

How it works
Step 1. Prepare the graph.
Step 2. Get the starting point and the final destination from the user.
Step 3. Apply BFS for searching the shortest way.
Step 4. Use DFS for exploring paths.
Step 5. Print results.

Possible applications
GPS and Google maps routing;
Planning delivery routes;
Movement of robots in warehouses;
Network routing;
Navigation of game characters.

Conclusion
This task allows us to study some basics of AI algorithmization. BFS guarantees the shortest path, while DFS allows to find an optimal way to explore nodes quickly.
